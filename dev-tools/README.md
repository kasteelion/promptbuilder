# Development Tools

Utilities for development, migration, and content generation.

## Structure

- **`migrations/`** - Data migration scripts for schema changes
- **`generators/`** - Content generation utilities (tags, summaries, etc.)
- **`validators/`** - Data validation and checking tools

## Common Scripts

### Migrations
- `migrate_characters.py` - Migrate character data format
- `migrate_outfits.py` - Migrate outfit structure
- `migrate_outfits_structure.py` - Restructure outfit directories

### Generators
- `generate_tags.py` - Auto-generate tags for content
- `generate_vibe_summaries.py` - Generate outfit vibe descriptions

### Validators
- `validate_color_schemes.py` - Validate color scheme definitions

### Other Tools
- `add_character_photo.py` - Add photos to character definitions
- `add_gender_tags.py` - Add gender tags to characters
- `analyze_outfit_overlap.py` - Find duplicate outfits
- `cleanup_outfit_bolding.py` - Clean up markdown formatting
- `scan_photos.py` - Scan for character photos

## Usage

These are development tools - not needed for normal app usage.
